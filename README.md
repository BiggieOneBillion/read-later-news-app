A robust, modern news discovery and management application. Built with **Next.js 14**, this app allows users to browse trending news and save articles to a personal dashboard for later reading.

## üöÄ Key Features

- **Dynamic News Fetching**: Real-time news discovery with an intuitive UI.
- **Personalized Dashboard**: Save, organize, and manage your favorite articles.
- **Secure Authentication**: User sign-up and login powered by Supabase/MongoDB.
- **Optimized Performance**: server-side rendering and client-side caching with React Query.
